ACT IV: How Color Emojis Actually Work

(The Technical Deep Dive)

Traditional fonts are simple: they store outlines of letters in black. Your system colors them as needed.But emojis are full-color, often with gradients, shading, and multiple layers. How do you store that?

There are three main approaches:

1. Bitmap Fonts (PNG)

Tiny PNG images stored inside the font file. Fast, but gets pixelated if you zoom in. Easy to implement but not scalable.

2. SVG Fonts (Vector)

Mathematical shapes. Scales infinitely without losing quality. Even zoom in 1000x, still sharp. Used by Mozilla/Adobe.

3. COLR/CPAL Tables

Layered shapes. Microsoft & Google use this. Each layer uses a specific color palette. Like a digital printing press.

Here is what the code looks like for the vector approach:

Layer 1: Face outline (yellow #FFCC4D)
Layer 2: Eyes (brown #664500)
Layer 3: Mouth curve (brown #664500)
Layer 4: Melting effect gradient (yellow to transparent)
